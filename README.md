# CryptoLab CTF

CryptoLab CTF is a full-stack-ready educational web app for classroom e-learning about cryptographic algorithms. It is inspired by modern security-training interfaces: dark lab UI, challenge cards, progress tracking, local CTF scoring, lecturer/student modes, and safe interactive labs.

The current MVP is frontend-only and stores progress in `localStorage`.

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Web Crypto API for SHA-256, HMAC-SHA-256, AES-GCM, and secure random values
- Manual TypeScript implementations for classroom/toy algorithms
- Vitest unit tests


## MVP Features

- Dashboard with category cards, score overview, progress, Start Lab actions, and mode toggles
- Algorithm Library with classical, symmetric, asymmetric, hashing, randomness, and trust modules
- Algorithm detail pages with explanation, formula, process, example input/output, strengths, weaknesses, status, and classroom activity
- Interactive labs:
  - Caesar encrypt/decrypt/brute force
  - Vigenere encrypt/decrypt and key-length signals
  - SHA-256 and avalanche comparison
  - HMAC-SHA-256 generator
  - AES-GCM encrypt/decrypt with Web Crypto
  - Toy RSA keygen/encrypt/decrypt/sign/verify
  - Toy Diffie-Hellman shared secret
  - Block cipher mode pattern-leakage visualization
- Cryptanalysis tools page for toy/classroom data only
- CTF challenge mode with hints, flags, score, and local validation
- Lecturer dashboard with flow suggestions, answer keys, worked solutions, talking points, reset, and progress export

## Classroom Usage

Suggested flow:

1. Start in Dashboard and switch between Student Mode and Lecturer Mode.
2. Open the Algorithm Library to introduce the concept, status, formula, and common mistakes.
3. Run the matching Interactive Lab on projector-friendly inputs.
4. Move to CTF Challenge Mode for short student exercises.
5. Use Lecturer Mode to reveal answer keys, reset local progress, or export progress JSON.

## Safety Scope

CryptoLab CTF is educational only.

Included cryptanalysis tools operate only on text entered in the app or intentionally weak toy examples generated by the app. The app does not include network scanning, exploit automation, password hash cracking, malware, phishing, credential theft, evasion logic, or tooling for attacking external systems.

Modern cryptography modules such as AES-GCM, RSA, ECC, HMAC, SHA-256, and key exchange explain real-world security boundaries. Real-world attacks against modern cryptographic systems are not implemented.

Weak or historical algorithms are clearly marked as `Educational only`, `Deprecated`, or `Broken`.
